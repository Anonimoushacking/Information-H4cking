¿Que es html 5?


Antes de empesar con el codigo debes tener claro que es el html 5 aca te dejo la informacion que debes saber



HTML5 es un lenguaje markup (de hecho, las siglas de HTML significan Hyper Text Markup Language) usado para estructurar y presentar el contenido para la web. Es uno de los aspectos fundamentales para el funcionamiento de los sitios, pero no es el primero. Es de hecho la quinta revisión del estándar que fue creado en 1990. A fines del año pasado, la W3C la recomendó para transformarse en el estándar a ser usado en el desarrollo de proyectos venideros. Por así decirlo, qué es HTML5 está relacionado también con la entrada en decadencia del viejo estándar HTML 4, que se combinaba con otros lenguajes para producir los sitios que podemos ver hoy en día. Con HTML5, tenemos otras posibilidades para explotar usando menos recursos. Con HTML5, también entra en desuso el formato XHTML, dado que ya no sería necesaria su implementación.



Teniendo esta pequeña idea de que es el html podemos empezar

Estructura basica

La estructura basica del html es:



 
<HTML>    
  <HEAD>  
    <TITLE> Título de la página </TITLE>
  </HEAD>  
  <BODY>  
    Contenido de la página
  </BODY>  
</HTML>    

Esa es la estructura basica de el html ahora te dejo las demas etiquetas y para que sirven.



  • La etiqueta <title> nos sirve para poner un titulo a nuestra pagina este titulo se vera en la pestaña de la pagina.

  • La etiqueta <head> esta nos permite poner elementos dentro de la cabecera de la pagina (dentro de esta va la etiqueta title o de lo contrario no funcionara)

  • La etiqueta <header> nos permite poner elementos en la cabeza de la pagina por ejemplo los h2 o los div.

  • La etiqueta <body> esta nos sirve para abrir el cuerpo de la pagina osea donde va a ir toda le informacion de la pagina.

  • La etiqueta <h1> sirve para un titulo de tamaño grande. Se divide desde < h1 > hasta <h6> que seria el tamaño mas pequeño al mas grande. Esto en cuanto a la etiqueta de titulos.

  • La etiqueta <div> Esta etiqueta nos permite crear una especie de caja o division que nos permite diferenciar varias partes de la pagina por ejemplo si la parte del titulo se encierra en un <div> y el contenido en otro se puede modificar en estilos de css cada uno por aparte cambiando asi el aspecto de una parte sin modificar la otra.

  • La etiqueta <ul> esta etiqueta nos señala que va a empezar una lista ordenada o desordenada la cual tambien se puede modificar con tus estilos de css.

  • La etiqueta <li> esta nos permite hacer una lista ordenada osea un menu desplegable o simplemente para ordenar contenido de un indice.

  • La etiqueta <section> esta etiqueta tiene un parecido con los div ya que tambien es para una seccion apartada pero esta se modifica junto al demas contenido.

  • La etiqueta <a> esta representa un enlace hacia una parte externa o una conexion en tu misma pagina.Para ello se ocupa la extencion href="" y dento de las comillas va tu enlace. Se usa para añadir tus redes sociales o simplemente para nevegar entre los documentos de tu pagina.

  • La etiqueta <script> esta etiqueta nos permite añadir un archivo de JavaScript que se encargara de las funciones de la pagina.

  • La etiqueta <footer> esta sirve para hacer el apartado de creditos o los derechos de autor.

Estas son las etiquetas basicas de debes usar siempre que desees crear una web basandote en el html espero sea de tu ayuda esta informacion y Gracias por visitar la pagina.
